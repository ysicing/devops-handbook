Drone配置管理员账号

November 11, 2021 · One min read

默认已经将drone数据持久化出来了

    spec:
      containers:
      - envFrom:
        - configMapRef:
            name: drone
        image: registry.cn-beijing.aliyuncs.com/k7scn/drone:2
        imagePullPolicy: Always
        name: drone
        volumeMounts:
        - mountPath: /data
          name: data
      restartPolicy: Always
      volumes:
      - hostPath:
          path: /data/drone
          type: DirectoryOrCreate
        name: data

debian安装sqlite3​

apt update
apt install sqlite3 -y

修改数据​

sqlite3 database.sqlite
.header on #开启头部显示。
.mode column # 列输出
.tables
select * from users;
update users set user_admin = 1;

验证​

drone user info ysicing
User: ysicing
Email: i@ysicing.me
Admin: true
Active: true
Machine: false
